## 🚀 Interactive Web App
I have deployed this model using **Streamlit**. You can interact with the model by adjusting customer parameters (tenure, charges, etc.) to see real-time churn predictions.

### How to run locally:
1. Clone this repo.
2. Run `pip install -r requirements.txt`.
3. Run `streamlit run app.py`.
